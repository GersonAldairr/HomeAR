# Trabajo grupal de IHC sobre nuestro proyecto HomeAR

Creado con Visual Studio Code, HomeAR es una plataforma digital orientada a mejorar la seguridad en viviendas autoconstruidas mediante el uso de Realidad Aumentada. La aplicación permite identificar posibles errores estructurales durante el proceso de construcción, brindando información visual y recomendaciones preventivas para reducir riesgos y promover edificaciones más seguras.

## Segmentos Objetivo

* Familias que realizan procesos de autoconstrucción de viviendas.
* Propietarios que desean verificar posibles riesgos estructurales en sus hogares.
* Estudiantes y técnicos del sector construcción interesados en herramientas de inspección asistida.
* Organizaciones y entidades que promueven viviendas seguras en comunidades vulnerables.

## Principales Características

* Detección visual de posibles fallas estructurales mediante Realidad Aumentada.
* Inspección guiada de elementos constructivos de la vivienda.
* Identificación temprana de riesgos que podrían comprometer la seguridad de la estructura.
* Interfaz intuitiva orientada a usuarios sin conocimientos técnicos especializados.
* Registro y consulta de inspecciones realizadas.
* Recomendaciones preventivas para mejorar la seguridad de la construcción.

## Tecnologías Utilizadas

* Visual Studio Code (editor principal de desarrollo)
* HTML5 para la estructura de la aplicación web
* CSS3 para estilos y diseño responsivo
* JavaScript para la interacción y funcionalidades dinámicas
* Git y GitHub para control de versiones y trabajo colaborativo
* Figma para el diseño de interfaces y prototipado
* GitHub Pages para el despliegue del prototipo web
